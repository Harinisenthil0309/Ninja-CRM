# Ninja CRM - Test Automation Suite

## Overview
Comprehensive test automation for Ninja CRM using Selenium WebDriver and TestNG. Covers contact, campaign, and product management modules with Page Object Model pattern and data-driven testing.

## Test Scenarios Covered

### Login Module
- Valid login with credentials
- Invalid login handling
- User authentication flows

### Contact Management
- Create new contacts
- Update contact information
- Search and filter contacts
- Data-driven contact scenarios

### Campaign Module
- Create campaigns
- Assign campaigns to contacts
- Campaign management workflows

### Product Module
- Create products
- Update product details
- Product inventory management

## Technologies Used
- **Selenium WebDriver** - Web automation
- **TestNG** - Test framework
- **Page Object Model (POM)** - Design pattern for maintainability
- **Data-Driven Testing (DDT)** - Excel-based test parameterization
- **Maven** - Build automation
- **Java** - Programming language
